<h2><h2><p style="font-family:monospace;text-align:center">Integration Guide<br></p></h2>
</th></tr>
</table>
<h3><p style="font-family:monospace;text-align:left"><strong>Version 1.0.1.0<br></strong></p></h3>
<p style="font-family:monospace">1. Extract downloaded .zip file<br>2. Open <code>src</code> folder<br>3. Merge <code>assets</code> folder with <code>assets</code> folder inside <strong>jsg config</strong> folder - location: <code>.minecraft\config\jsg</code></br>4. Launch Minecraft with <strong>Forge and JSG</strong></br>5.  Join any world and run command <code>/jsg originsimport</code> to import new origins<br>    * This command is server-side - on multiplayer will be changed <strong>ONLY</strong> server side config<br>    * This command require permission level 2 and heigher<br><br></p>

<h3><p style="font-family:monospace;text-align:left"><strong>Version 2.0.0.0 +<br></strong></p></h3>
<p style="font-family:monospace">1. Extract downloaded .zip file<br>2. Open <code>src</code> folder<br>3. Merge* <code>assets</code> folder with <code>assets</code> folder inside <strong>minecraft</strong> folder - location: <code>.minecraft\assets</code></br>    * By merging we mean copy over the <code>jsg_origins</code> folder into your <code>assets</code>> folder</p>
